# Warehouse Inventory Management System

**Description**
This is a simple Java program to manage warehouse items.
It allows users to add, remove and display items.

**Features**
- Add item
- Remove item
- Show all items

**How to Run**
1. Compile:
   javac *.java

2. Run:
   java Warehouse

**Sample Output**
